CODE 23 FIRAT AKADEMİ TAKVİMİ

Code 23 Fırak Akademi Takvimi 01 Haziran 2023

Code 23 Fırat Akademi Takvimi Sanayi ve Teknoloji Bakanlığı Kalkınma Ajansları Genel Müdürlüğü, Fırat Kalkınma Ajansı tarafından Fırat Üniversitesi 2022 yılı Cazibe Merkezi Destekleme Programı (CMDP) kapsamında Elazığ'da uygulanması planlanan “CODE-23 Fırat Yazılım Atölyesi” projesinin 2023 Akademik Takvimi yayınlanmıştır. Bölgenin üretim potansiyelini artırmak ve teknoloji alanında öncü bireyler yetiştirmek amacıyla Code 23 Fırat Eğitim Akademisi 2023-2024 eğitim dönemi için başlangıç tarihi, 15 Haziran 2023 günüdür. Bu dönemde, gençlerin yeteneklerini ve potansiyellerini ortaya çıkarmak adına Code 23 Fırat Eğitim Akademisi’nin içerdiği zengin eğitim programlarıyla gençlerin kariyer yollarında emin adımlarla ilerlemeleri için destek olunacaktır. Code 23 Fırat Eğitim Akademisi Takvimi, eğitim programlarının yanı sıra atölye çalışmaları, seminerler ve diğer etkinlikleri içermektedir. Bu takvimde, seçkin eğitmenlerimizin liderliğinde yazılım geliştirme, veri bilimi, yapay zeka, web programlama, siber güvenlik, mobil uygulama geliştirme gibi alanlarda kapsamlı dersler sunulacaktır. Ayrıca, takvimimizde yer alan etkinlikler ve atölye çalışmaları, katılımcılarımızın birbirleriyle etkileşim kurarak deneyimlerini paylaşmalarını ve işbirlikleri geliştirmelerini hedeflemektedir. Bu etkinlikler, bölge gençlerimizin girişimcilik ruhunu teşvik edecek ve teknik becerilerini gerçek dünya uygulamalarıyla pekiştirecek bir platform sunacaktır. Bu benzersiz eğitim ve gelişim fırsatlarından yararlanmak için https://code23.firat.edu.tr adresini takip etmeyi unutmayınız.
